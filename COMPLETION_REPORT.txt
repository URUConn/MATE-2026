🚀 QUICK START GUIDE

Installation (30-60 minutes)
  1. On both LattePanda and Control Laptop:
     mkdir -p ~/mate_rov_ws/src
     cd ~/mate_rov_ws
     git clone <your-repo> src/
     bash src/install.sh
     source install/setup.bash
     export ROS_DOMAIN_ID=42

  2. Configure PIX6 in QGroundControl:
     • SERIAL2_PROTOCOL = 38 (MAVLink v2)
     • SERIAL2_BAUD = 115200
     • Set up servo channels 1-6 for thrusters

  3. On LattePanda, launch onboard system:
     ros2 launch rov_onboard onboard_launch.py mavlink_connection:=/dev/ttyUSB0:115200

  4. On Control Laptop, launch control station:
     ros2 launch rov_control control_launch.py

  5. Verify system:
     ros2 topic echo /rov/status
     ros2 topic echo /rov/sensor_data

Testing (1-2 hours)
  Follow VERIFICATION_CHECKLIST.md step-by-step before field deployment

🔧 CONFIGURATION REFERENCE

Connection String Examples:
  • Serial: /dev/ttyUSB0:115200
  • Serial alternate: /dev/ttyUSB1:57600
  • UDP: udp:192.168.1.100:14550

Launch Examples:
  # Default configuration
  ros2 launch rov_onboard onboard_launch.py

  # Custom serial port
  ros2 launch rov_onboard onboard_launch.py mavlink_connection:=/dev/ttyUSB1:115200

  # UDP connection
  ros2 launch rov_onboard onboard_launch.py mavlink_connection:=udp:192.168.1.100:14550

Configuration File: src/rov_mavlink/config/mavlink_params.yaml
  - connection_string: Serial or UDP connection
  - system_id: PIX6 system ID (usually 1)
  - component_id: Component type (191 = Submarine)
  - heartbeat_interval: Heartbeat frequency in seconds
  - telemetry_rate: Telemetry publish rate in Hz
  - servo_channel_mapping: [ch1, ch2, ch3, ch4, ch5, ch6] for 6 thrusters
